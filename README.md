# Portfolio

Personal website powered by React, Vite, and Tailwind CSS.

## Getting started

```bash
npm install
npm run dev
```

Visit `http://localhost:5173/` for live reload during development.

## Structure

- `src/components/PersonalSite.jsx` – main page component.
- `src/index.css` – Tailwind base styles and fonts.
- `src/main.jsx` – app entry point.
- `public/img` – static assets like profile photos.

Run `npm run build` to generate the production bundle in `dist/`.
